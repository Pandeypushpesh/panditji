# Images Folder

Please place Pandit Ji's photo in this folder with the filename:

**pandit-ji-photo.jpg**

## Image Requirements:
- Recommended size: 600x600 pixels or larger (square format works best)
- File formats: JPG, JPEG, or PNG
- The image will be displayed in a circular frame, so a square or portrait orientation works best

## Alternative Filenames:
If you use a different filename, please update the `src` attribute in `index.html` at line 43:
```html
<img src="images/your-filename.jpg" alt="पंडित निरंजन शास्त्री" class="pandit-photo">
```
